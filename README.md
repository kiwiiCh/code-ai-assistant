# 🚀 No-Limit Code AI

A mobile-optimized, Claude-style coding assistant with zero restrictions. Internet-aware, downloadable files, copyable code — built for developers who want full freedom.

## ✨ Features

- 💬 **Claude-inspired UI** — clean chat interface, mobile-first
- 🌐 **Internet-enabled** — simulated web access for better coding context
- 📋 **Copy & Download** — every code response comes with one-click copy and file download
- 🔓 **No restrictions** — no content filters, no barriers
- ⚡ **Pure frontend** — no backend, host anywhere

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, custom properties)
- Vanilla JavaScript (ES6+)
- Font Awesome icons

## 📦 Quick Deploy

### Option 1: Render (Static Site)
1. Push this repo to GitHub (public)
2. Go to [render.com](https://render.com) → New → Static Site
3. Connect repo, set **Publish Directory** to `.`
4. Deploy — live in 30 seconds

### Option 2: GitHub Pages
1. Settings → Pages → Source: `main` branch → `/ (root)`
2. Save → live at `username.github.io/repo-name`

### Option 3: Run Locally
```bash
git clone https://github.com/your-username/code-ai-assistant.git
cd code-ai-assistant
